
# Assignment 6 – Linked List Variants

This assignment includes implementations for multiple linked list structures:

## 1. Circular Linked List (Full Menu Program)
Supports:
- Insert at beginning
- Insert at end
- Insert before given position
- Delete a node
- Search a node
- Display the list

## 2. Circular Linked List (Head repeated and non‑repeated display)
- Inserts elements at end
- Displays circular list normally
- Displays circular list with head repeated at end

## 3a. Doubly Linked List – Size of List
- Inserts at end
- Displays list
- Computes size

## 3b. Circular Linked List – Size of List
- Inserts at end
- Displays list
- Computes count using circular traversal

## 4. Doubly Linked List – Palindrome Check
- Inserts characters
- Checks if list reads same forward and backward

## 5. Detect Circular Linked List
- Uses slow–fast pointer technique
- Prints normal and circular lists
- Checks whether list is circular

## How to Compile and Run
```
g++ filename.cpp -o output
./output
```
